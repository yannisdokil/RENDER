# Lampac + lampac-ukraine on Render.com

## Деплой

1. Завантаж (upload) цю папку на GitHub або інший Git-сервер.  
2. На Render.com створіть Web Service → Docker → Port = 9118.  
3. Render автоматично побудує контейнер та запустить Lampac.  
4. Відкрий даний URL — Lampac з українськими модулями готовий 😊

## Модулі

- AnimeON  
- Anihub  
- Unimay  
- CikavaIdeya  
- Uaflix  
- UaTUT
